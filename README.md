[MTProxy for Telegram](https://github.com/shellhub/blog/issues/12)

``` bash
wget -N --no-check-certificate https://raw.githubusercontent.com/shellhub/shellhub/master/proxy/mt_proxy.sh && chmod +x mt_proxy.sh && ./mt_proxy.sh
```
